# 🌟 EchoMiroirBot

![Banner](https://files.catbox.moe/oziw3x.png)

🤖 Un bot Telegram avec 35+ commandes (Texte, Jeux, Calculs, Fun).

👤 **Créé par HAENYLTR**

---

## 🚀 Commandes

- **Texte :** /echo, /caps, /low, /cap, /rev, /mix, /spoil, /len
- **Calculs :** /calc, /roll, /random
- **Jeux :** /dice, /coin, /roulette, /joke, /quote, /choose
- **Temps :** /time, /date, /weather
- **Utilitaires :** /ping, /cdown, /stats, /uid, /uptime
- **Fun :** /flip, /emoji, /pyramide, /morse, /encrypt, /decrypt
